Data Visualisations

Ebola 3W Dashboard

A dashboard with interactive charts to help Ebola responders understand and analyse data from the the UN OCHA Who, What, Where data set. This dashboard was hosted on the UN run Humanitarian response website as one of the main tools for coordinating response efforts.

Ebola Timeline Map

This interactive map was designed to help understand the week by week situation of the West Africa Ebola crisis. Other visualisations were using cumulative totals where this differed by showing snapshots highlighting the trend of the spread over time.

Ebola Narrative

A speculative design where the traditional narrative story is still the focus and the data visualisation acts as an accompaniment to enhance the narrative rather than the being the focus. This visualisation was modified and published in the Swiss newspaper Le Temps.

Ebola Dashboard

Produced as part of the IFRC response this dashboard became the main method for disseminating information during the Red Cross efforts to combat the Ebola outbreak.

Typhoon Haiyan 3W Dashboard

UN OCHA's Who, What, Where interactive dashboard hosted on the Humanitarian Response website.

How MPs voted

A data visualisation to facilitate exploration of trends in how political parties voted on the controversial UK Data Retention Bill.
